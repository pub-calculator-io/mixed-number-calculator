Mixed number calculator is designed to help you add, subtract, multiply, and divide mixed numbers. Mixed Fractions Calculator is created for performing mathematical operations on mixed numbers.

[https://www.calculator.io/mixed-number-calculator/](https://www.calculator.io/mixed-number-calculator/)

== Usage ==

Add the Mixed Number Calculator shortcode to your page, post or sidebar:

`[ci_mixed_number_calculator]`

Add the following code to your template where you would like the Mixed Number Calculator to appear:

`<?php display_ci_mixed_number_calculator(); ?>`

1. Upload the Mixed Number Calculator /ci_mixed_number_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Mixed Number Calculator plugin through the "Plugins" menu in WordPress.
